# BIOBAT

Simple Web application to manage registrations of organic-certified players (LBB-like).

## Fork

BIOBAT is a fork of the BMF-suivi app which has not been released previously.

## Notes

### Entités et cie

- Joueurs
  - Réguliers
  - Occasionnels
  - Liste des personnes
- Matchs
  - Événement FB
  - Liste et nombre
  - Date
- Saisons
  - Frais/saison
  - Frais/matchs
  - Even

### Interface

- Couleur dans la liste : Payé / À percevoir, largeur d'écran
- Configurer la saison
